# Shivansh Silk Mill

A React + Vite website for Shivansh Silk Mill, a premium shirting and garment fabric manufacturer.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite, usually `http://127.0.0.1:5173/`.

## Checks

```bash
npm run lint
npm run build
```

## Main Files

- `src/pages/Home.jsx` controls the home page section order.
- `src/components/` contains the site sections.
- `src/index.css` contains the full site styling.
- `src/assets/` contains local images and the hero video.
